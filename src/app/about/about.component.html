About GitSearch

GitSearch is a website that uses Git Api to display the profile and the repositories of a person has

Through this, one can enter their github username and click search to view their profile and repositories

I have managed to build this using the following languages:

Angular
TypeScript
HTML
CSS & Bootstrap
Run and Watch Tests
Build for Production
